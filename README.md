# Coach's Log

Веб-приложение для учета групп, тренировок, посещаемости и оплат тренера.

## Стек
- Python
- FastAPI
- SQLAlchemy
- SQLite

## Реализовано
- CRUD для групп
- CRUD для учеников
- CRUD для тренировок
- Журнал посещаемости по тренировке

## Запуск
pip install -r requirements.txt
uvicorn app.main:app --reload
